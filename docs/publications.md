# Publications

A complete list of research articles I am involved in can be found on my [Google Scholar profile](https://scholar.google.com/citations?user=zz6G_wsAAAAJ&hl=en).

Contributions to software and websites can be found in my [GitHub repository](https://github.com/DunklesArchipel) or the [echemedb organization](https://github.com/echemdb), including

* [echemdb website](https://www.echemdb.org) provides an overview on echemdbs' activities and contains a database for electrochemical cyclic voltammetry data
* [unitpackage](https://echemdb.github.io/unitpackage/) is a Python API to interact with frictionless datapackages, for example, used for the creation of the echemdb website [![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.7942320.svg)](https://doi.org/10.5281/zenodo.7942320)
* [svgdigitizer](https://echemdb.github.io/svgdigitizer/) is a Python library and command line tool to recover the measured data underlying plots in scientific publications [![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.5881475.svg)](https://doi.org/10.5281/zenodo.5881475)
