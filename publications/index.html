Publications

Below you find references and links to software packages and datasets I was (or still am) involved in, which play an important part in modern research and often do not receive as much attention as a Journal Article (at least in some research areas such as chemistry).

Contributions to software and websites can be found in my GitHub repository or the echemdb organization.

Software¶

Apart from the packages listed below, additional tools for data curation, annotation, and manipulation can be found in the echemdb open-source community.

  • echemdb website provides an overview of echemdb’s activities and contains a database for electrochemical cyclic voltammetry data (Engstfeld et al. (2022))

  • unitpackage is a Python API to interact with frictionless datapackages, used for example in the creation of the echemdb website (Engstfeld et al. (2023))

  • svgdigitizer is a Python library and command line tool to recover the measured data underlying plots in scientific publications (Engstfeld et al. (2022))
