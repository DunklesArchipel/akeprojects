## Software

Apart from the packages listed below, additional tools for data curation, annotation, and manipulation can be found in the [echemdb open-source community](https://github.com/echemdb).

* [echemdb website](https://www.echemdb.org) provides an overview of echemdb's activities and contains a database for electrochemical cyclic voltammetry data ({cite}`echemdb_website_2022`)
* [unitpackage](https://echemdb.github.io/unitpackage/) is a Python API to interact with frictionless datapackages, used for example in the creation of the echemdb website ({cite}`unitpackage_2023`) [![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.7942320.svg)](https://doi.org/10.5281/zenodo.7942320)
* [svgdigitizer](https://echemdb.github.io/svgdigitizer/) is a Python library and command line tool to recover the measured data underlying plots in scientific publications ({cite}`svgdigitizer_2022`) [![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.5881475.svg)](https://doi.org/10.5281/zenodo.5881475)
